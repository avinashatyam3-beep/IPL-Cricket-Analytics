# IPL Cricket Analytics Project

## Overview
This project analyzes IPL cricket data using Python and Power BI.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn


## Analysis Done
- Team Wins Analysis
- Player of the Match Analysis
- Toss Impact Analysis
- Top Batsmen Analysis
- Top Bowlers Analysis
- Venue Analysis
- Strike Rate Analysis
- Economy Rate Analysis

## Project Files
- main.py - Python analysis code
- data/ - IPL dataset files
- output/ - Processed CSV files

## Insights
- Top IPL teams were identified based on total wins.
- Top batsmen and bowlers were analyzed using ball-by-ball data.
- Toss impact and venue trends were studied.
- Strike rate and economy rate were calculated for performance analysis.

## Conclusion
This project helped understand real-world cricket analytics using Python.
